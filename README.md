# Power BI Analyst

Repositório relacionado aos módulos de Power BI Analyst da formação Python Data Analytics da DIO. Os desafios dentro desse módulo são

| Pasta                    | Descrição                                                                                                                                                                                                              |                      Status                      |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------: |
| [Desafio 1](<Desafio 1>) | Cria um relatório no Power BI usando a amostra de dados financeiros.                                                                                                                                                   | ![Done](https://img.shields.io/badge/Done-green) |
| [Desafio 2](<Desafio 2>) | Cria uma database no Azure, conectar e inserir a amostra de dados. Usar o Power BI para conectar na database e realizar o processo de transformação e carregamento. Usar os dados para criar um relatório no Power BI. | ![Done](https://img.shields.io/badge/Done-green) |
